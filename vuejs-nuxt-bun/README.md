# Vue.js + Nuxt + Bun Plugin

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A plugin providing TypeScript LSP with Vue.js support for Bun (instead of Node+npm). Includes Vue.js, Nuxt, Nuxt UI, and VueUse skills.

**Features:**

- TypeScript LSP with Vue.js support (using `bunx`)
- Vue 3 Composition API patterns and best practices
- Nuxt 3 conventions and utilities
- Nuxt UI component library guidance
- VueUse composable references

**Supported file types:** `.vue`, `.ts`, `.tsx`, `.js`, `.jsx`, `.mts`, `.cts`, `.mjs`, `.cjs`

## Available Skills

| Skill     | Description                                                     |
| --------- | --------------------------------------------------------------- |
| `vue`     | Vue 3 Composition API patterns, component architecture, testing |
| `nuxt`    | Nuxt 3 conventions, routing, server routes, middleware          |
| `nuxt-ui` | Nuxt UI component library usage and styling                     |
| `vueuse`  | VueUse composables and utilities                                |

## Requirements

- [Bun](https://bun.sh/) runtime
- Vue.js 3.x project
- (Optional) Nuxt 3.x for Nuxt-specific features
- (Optional) Nuxt UI for UI component guidance
